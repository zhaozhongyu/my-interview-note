start() 

启动程序并返回一个 Promise 保证当start()结束时程序是可用的. You should always wait for start to complete before running any commands.

stop()

停止程序并返回 Promise 对象.

restart()

重启程序并返回 Promise 对象.

isRunning()

检测程序是否在运行.

返回 Boolean值.

getSettings()

Get all the configured options passed to the new Application() constructor. This will include the default options values currently being used.

Returns an Object.
