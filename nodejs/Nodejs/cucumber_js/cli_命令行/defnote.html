执行全部用例时./node_modules/.bin/cucumber-js
  • 正则表达式执行
    • $ cucumber-js features/**/*.feature
  • 执行整个目录
    • $ cucumber-js features/dir
  • 执行单个文件
    • $ cucumber-js features/my_feature.feature
  • 执行单个文件中的单个用例, 输入的是行数
    • $ cucumber-js features/my_feature.feature:3
  • 在所有文件中查找对应名字执行(只能匹配一个)
    • $ cucumber-js --name "topic 1"
    • If used multiple times, the scenario name needs to match only one of the names supplied

--format-options "{\"colorsEnabled\":false}"  不使用颜色


os.system('node_modules\.bin\cucumber-js --format-options "{\\"colorsEnabled\\":false}" features\spectron\spec.feature ')
