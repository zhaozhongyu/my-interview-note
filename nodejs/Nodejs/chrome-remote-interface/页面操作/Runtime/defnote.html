运行时处理, 核心方法

Runtime.awaitPromise#

Add handler to promise with given promise object id.

Runtime.discardConsoleEntries#

Discards collected exceptions and console API calls.

Runtime.evaluate#核心方法, 用于执行js

Evaluates expression on global object.

执行普通命令行参数时, Runtime.evaluate({expression: 'document.documentElement.outerHTML'});

执行函数方法时: Runtime.evaluate({ expression: `(${demo})()` });

returnByValue: true表示安装json格式返回

awaitPromise: true 表示等待promise执行完成


Runtime.runScript#

Runs script with given id in a given context.
